Projects

FTC Power Play - Java Programming

For my involvement in the First Tech Challenge (FTC) Power Play, I led the development of Java programs for both the autonomous and manual sections of the challenge. These programs played a crucial role in the successful execution of our team's robot during competitions.

The autonomous program were designed to make the robot perform predefined tasks without direct human control, showcasing precision and efficiency. The manual control programs allowed our team to operate the robot seamlessly during teleoperated periods, contributing to our overall strategy and performance.

The codebase for this project is available on GitHub.

Ion Inc. - Congressional App Challenge

Ion Inc. is an educational app created using block code for the 2021 Congressional App Challenge in district NJ-03. The purpose of this app is to teach kids about the various kinds of renewable energy in an entertaining way. Through interactive and engaging content, Ion Inc. aims to make learning about renewable energy sources a fun and enjoyable experience for young minds.
